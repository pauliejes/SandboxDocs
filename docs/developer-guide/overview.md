# Overview



Assuming you've installed [Node.js](http://nodejs.org/dist/latest-v0.12.x/), here is the 1 minute install:

```
> git clone https://github.com/adlnet/Sandbox.git
> npm install
> cd Sandbox
> node app.js
> browse to http://localhost:3000
```
